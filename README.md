# swaga666
for (cool) nerds only
AI-АГРЕГАТОР НОВОСТЕЙ

гипотеза работы:
Когда я хочу быстро понять суть важных мировых событий и как о них думают разные источники, я хочу получить структурированную карту мнений и событий, чтобы формировать своё беспристрастное мнение без необходимости читать множество статей/новостей

Ценность будущего MVP:
Это будет мобильное приложение, в котором пользователи смогут ознакомиться со сводкой новостей из различных источников, авторитетных СМИ по выбранному событию. Пользователи смогут быстро получить беспристрастную картину происходящего и выйти из информационного пузыря

лента “событий”:
 • “Выборы в США”
 • “Конфликт X”
 • “Новый закон в ЕС”

карта мнений внутри каждого события:
 • как разные источники освещают событие
 • где расходятся
 • где сходятся

что конкретно внутри:
 • краткое summary
 • ключевые факты
 • сводка мнений от различных изданий

Потенциальная целевая аудитория (гипотеза):
Активные потребители новостей. Люди, которые хотят быть в курсе мировых событий, но не могут читать десятки статей ежедневно.
